# MJOQBOShear
Code for Roundy 2026 in Climate, role of vertical shear of the zonal wind in the MJO/QBO relationship.

speedregqboulong.py analyzes monthly reconstructed MJO RMM index and standard RMM index amplitude data for its associations with 
the monthly 70 hPa QBO zonal wind data and the 70-200 hPa zonal wind shear. 
speedregqboseasensoio.py analyzes the 1974-2020 daily ERA5 and RMM index data to replicated figures in Roundy 2026 MJO/QBO in Climate. 
speedregqbovertglobshear.py finds vertical cross sections of daily equatorial zonal wind and geopotential height data from the ERA5 reanalysis
for comparing QBO eastelry and westerly and 70-200 hPa shear of the zonal wind (easterly and westerly) replicating Figure 7. The statistical test won't exactly replicate (as in any bootstrap or Monte Carlo experiment), but will replicate in bulk.
